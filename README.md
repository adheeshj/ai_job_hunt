## AI-Powered Government Job Application Automation System

Developed an end-to-end multi-agent AI system that automates the government job application workflow using CrewAI, LangChain, Google Gemini, and Streamlit.

Designed and implemented specialized AI agents with distinct responsibilities, including a Job Analysis Agent for extracting qualifications, skills, and eligibility requirements from job postings, a Resume Customization Agent for tailoring resumes to specific positions, and a Professional Messaging Agent for generating personalized cover letters, networking messages, and LinkedIn outreach.

Built a collaborative CrewAI workflow where agents operate autonomously and pass structured outputs between one another, creating an intelligent application pipeline that transforms raw job listings into customized application materials.

Integrated the USAJobs API to fetch real-time federal government job opportunities and provide live data for agent processing. Implemented automated requirement extraction, keyword analysis, and qualification matching to improve application relevance and ATS compatibility.

Developed a Streamlit-based web application that allows users to search and filter job opportunities, review analyzed listings, select target positions, and generate customized resumes, cover letters, and professional outreach messages with a single click.

Implemented persistent application tracking, logging, and document storage to maintain a history of generated materials and submitted applications. The system demonstrates practical agent orchestration, API integration, LLM-powered automation, and multi-agent collaboration for real-world career assistance workflows.
